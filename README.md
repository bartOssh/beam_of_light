### Dependencies

- [Pytorch](https://pytorch.org/)
- [Flask](https://flask.palletsprojects.com/en/1.1.x/)
- [Flask Api](https://www.flaskapi.org/)
- [Matplotlib](https://matplotlib.org/)

### Instalation

- This project uses Python 3.6 or greater
- Install virtual environment: [venv](https://packaging.python.org/guides/installing-using-pip-and-virtual-environments/)
- Set source to venv, linux example: `source venv/bin/activate`
- Install requirements, linux example: `pip install -r requirements.txt`
- Run training / testing / local detection session `python detect.py -h` to get help how to use it.
